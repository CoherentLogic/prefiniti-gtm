2 Object Record Management System

2.0.1 Records (DWORREC)

2.0.1.1 $$CREATE^DWORREC(PAR,CLS,NAM,KEY,PDB) - Create Record
2.0.1.2 $$OUTPUT^DWORREC(OID,KEY,PDB) - Revise Record's PDB
2.0.1.3 $$APPEND^DWORREC(OID,KEY,PDB) - Add Key/PDB To Record
2.0.1.4 $$INPUT^DWORREC(OID,KEY,REV) - Read Record's PDB by OID/key/revision
2.0.1.5 $$CLSGET^DWORREC(CLASNAME,KEY,PDB) - Get PDB through class/key/PDB
2.0.1.6 $$INDGET^DWORREC(CLASNAME,KEY,PDB) - Get OID through class/key/PDB
2.0.1.7 $$NEWEST^DWORREC(OID,KEY) - Get Highest Rev. # through OID/key
2.0.1.8 SETLOCK^DWORREC(OID) - Lock record by OID
2.0.1.9 $$ISLOCKED^DWORREC(OID) - Is record OID locked?
2.0.1.10 CLRLOCK^DWORREC(OID) - Clear lock on record OID
2.0.1.11 $$ROOTNODE^DWORREC() - Get Root Node's OID
2.0.1.12 SETRTNOD^DWORREC(OID) - Set Root Node to OID
2.0.1.13 $$RECEXIST^DWORREC(OID) - Does record OID exist?
2.0.1.14 DUMP^DWORREC(OID) - ZWR record OID
